Angular Work

I have been working with Angular from version 1.3 and up. Below are some of the projects I have worked on. In fact this page is also an Angular 5 app. Feel free to check out the code at the Github repo.

Portal App

A portal application for users to access information from multiple systems. The frontend was fed from a Redis cache to increase the application's speed. The cache was populated from cron jobs nightly pulling data from multiple systems.

CMS App

A CMS that controls information being pushed out to a native mobile app for in house training in a large company. The CMS controlled user management, slot allocation, plus data that the native phone Android/IOS app would display.

User Allocation App

An application for allocating users to homes based on certain criteria. Contains user management plus multiple roles and profiles. API data is fed from a Symfony2 backend with Postgres database.

Wedding Planner App

An Angular 5 application for users to manage their wedding. Built with Node and MongoDB backend. Client side state is managed using ngrx store and ngrx effects. This is a personal project and is currently in development. Feel free to have a look at the Github repo
